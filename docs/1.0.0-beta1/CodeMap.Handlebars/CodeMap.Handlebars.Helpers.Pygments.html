Pygments Class

A helper used to generate a code block having highlighted syntax, this helper is based on PygmentSharp.

Base type: object.
Implemented interfaces: IHandlebarsHelper.

Example

The following template will generate a code block containing C# highlghed code. 

{{Pygments 'public class MyClass { }' 'c#'}}
