CodeMap.Handlebars.Helpers Namespace

Contains the Handlebars.NET helpers that are used in the default Handlebars templates.

Interfaces

Name Summary
IHandlebarsBlockHelper

Represents a black Handlebars helper.

IHandlebarsHelper

Represents an inline Handlebars helper.

Classes

Name Summary
Concat

A concatenation helper, concatenates each provided parameter.

Format

A format helper, useful for applying different formats to a value.

GetAssemblyCompany

A helper used to get the company name from an AssemblyDeclaration.

GetDocumentationPartialName

A helper used to get the partial name for a DocumentationElement.

IsCollection

A helper used to determine whether a value is a collection.

MemberAccessModifier

A helper used to get the access modifier textual description for a DeclarationNode. Most useful for properties where the access modifier can be changed for one of the accessors.

MemberReference

A helper used to generate hyperlinks for a DeclarationNode or a MemberReference.

Pygments

A helper used to generate a code block having highlighted syntax, this helper is based on PygmentSharp.

Semver

A helper used to convert a Version into a Semver expression.
