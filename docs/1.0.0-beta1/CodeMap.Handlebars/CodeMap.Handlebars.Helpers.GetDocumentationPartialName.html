GetDocumentationPartialName Class

A helper used to get the partial name for a DocumentationElement.

Base type: object.
Implemented interfaces: IHandlebarsHelper.

Example

The following template will generate a paragraph containing the partial name for a DocumentationElement. 

<p>{{GetDocumentationPartialName}}</p>

If the current context is an SummaryDocumentationElement, the output would be equal to Summary (Summary): 

<p>Summary</p>

While it may not be useful to display this information like this, it is useful to use this helper as a selector for a specific partial. For instance, the Documentation partial is defined like this: 

{{#if (IsCollection node)}}
{{#if asList}}
{{#each node}}
{{#if @first}}
{{#if ../title}}<h3>{{../title}}</h3>{{/if}}
<ul>
    {{/if}}
    <li>{{> (GetDocumentationPartialName this)}}</li>
    {{#if @last}}
</ul>
{{/if}}
{{/each}}
{{else}}
{{#each node}}
{{#if @first}}{{#if ../title}}<h3>{{../title}}</h3>{{/if}}{{/if}}
{{> (GetDocumentationPartialName this)}}
{{/each}}
{{/if}}
{{else if node}}
{{#with node}}{{> (GetDocumentationPartialName) title=../title}}{{/with}}
{{else if (IsCollection this)}}
{{#each this}}{{> (GetDocumentationPartialName)}}{{/each}}
{{else}}
{{> (GetDocumentationPartialName)}}
{{/if}}

As you can see, the specific DocumentationElement partial name is selected using this helper, it's like determining the name of the function you want to call dynamically.
