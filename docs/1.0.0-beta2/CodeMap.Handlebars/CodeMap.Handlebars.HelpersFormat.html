Format Class

A format helper, useful for applying different formats to a value.

Base type: object.
Implemented interfaces: IHandlebarsHelper.

Constructors

Name Access Modifier Summary
Format() public

Properties

Name Access Modifier Summary
Name public get

Gets the name of the helper.

Methods

Name Access Modifier Summary
Apply(TextWriter, object, object[]) public

Formats the first parameter which is expected to be an IFormattable by applying the format string provided through the second parameter which is expected to be a string.

Example

The following template will generate a paragraph containing the formatted value. 

<p>{{Format this '0.00'}}</p>

If the current context is a number, say 123 the output would be: 

<p>123.00</p>
